# MA // Migration Analytics 🌍✈️

O **MA // Migration Analytics** é um painel tático e interativo de planejamento migratório desenvolvido sob medida para guiar a nossa jornada internacional em casal.

Este projeto nasceu de uma sessão descontraída de **Vibe Coding** — focada em transformar dados, planos e estratégias de imigração em uma ferramenta visual, dinâmica e extremamente útil para os nossos objetivos de vida e carreira em 2026.

## 🚀 O Cenário Planejado

O painel foi parametrizado especificamente para um perfil de alta demanda e foco estratégico:

* **Área de Atuação:** Tecnologia da Informação (TI) em transição/estudos, focado em suporte técnico e infraestrutura.
* **Proficiência:** Inglês avançado (C1).
* **Visto:** Busca ativa por patrocínio corporativo ( *Sponsorship* ).
* **Casal:** Planejamento conjunto com direitos de trabalho para o cônjuge.
* **Destinos Analisados:** 🇮🇪 Irlanda (Plano A), 🇳🇿 Nova Zelândia (Plano B), 🇦🇺 Austrália (Plano C) e 🇨🇦 Canadá (Plano D).

## 🛠️ Funcionalidades Principais

* **Dossiê Comparativo Dinâmico:** Abas interativas para explorar os requisitos, burocracia, cronogramas de visto e estratégias de carreira de cada país.
* **Calculadora de Custos Adaptativa:** Baseada nos orçamentos mínimos reais de cada país, convertendo automaticamente as moedas locais (`EUR`, `NZD`, `AUD`, `CAD`) para o **Real Brasileiro (R$)** em tempo real usando taxas de câmbio atualizadas via API.
* **Margem de Emergência Interativa:** Ajuste dinâmico de 0% a 50% para prever variações de custo e taxas de segurança.
* **Rastreador Semanal de Metas:** Um painel tático para acompanhar as 125 aplicações semanais (vagas/dia divididas por país de destino).
* **Monitoramento Ativo de Empresas:** Pipeline de candidaturas focado nas empresas recomendadas com *watermarks* premium e gerenciamento de status ( *Aplicado* ,  *Entrevista* ,  *Aprovado* ).
* **Termos de Busca com Copiar Rápido:** Atalho para copiar as melhores *strings* de pesquisa para colar nos buscadores de vagas.

## 💻 Tecnologias Utilizadas

Para manter o projeto extremamente leve, rápido de carregar e fácil de hospedar, toda a aplicação foi construída em um  **único arquivo estático** :

* **HTML5 & CSS3** (Estrutura e semântica moderna).
* **Tailwind CSS** (Estilização responsiva em modo escuro premium).
* **JavaScript (ES6+)** (Lógica da calculadora, controle de abas e manipulação de estado local).
* **ExchangeRate-API** (Consumo de câmbio de moedas comerciais ao vivo com sistema de contingência offline).
